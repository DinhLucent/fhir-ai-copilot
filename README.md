> Python CLI copilot for FHIR R4 resources: smart validation, resource templates, HL7-to-FHIR conversion, and natural language query helpers

## ✨ Features

- ✅ Python CLI copilot for FHIR R4 resources: smart validation, resource templates, HL7-to-FHIR conversion, and natural language query helpers

## 📦 Installation

```bash
git clone https://github.com/DinhLucent/fhir-ai-copilot.git
cd fhir-ai-copilot
pip install -r requirements.txt
```

## 🚀 Quick Start

```bash
python src/main.py
```
